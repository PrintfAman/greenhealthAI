# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive in technical discussions.
- Assume good intent and focus on evidence-based feedback.
- Keep communication professional and inclusive.

## Unacceptable Behavior

- Harassment, discrimination, personal attacks, or intimidation.
- Publishing private information without consent.
- Disruptive behavior that blocks collaboration.

## Enforcement

Project maintainers may remove or edit comments, issues, or PRs that violate this code of conduct.

## Scope

This policy applies in project spaces such as issues, pull requests, commit discussions, and chat channels used for project collaboration.
