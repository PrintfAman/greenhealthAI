# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-02-26

### Added

- Pathway-based streaming telemetry ingestion and aggregation pipeline.
- FastAPI APIs for metrics, score, alerts, health probes, and copilot.
- Pathway xPack RAG server and copilot integration.
- React dashboard with live charts and copilot UI.
- Docker Compose local runtime.
- Deployment support for Render (backend) and Vercel (frontend).
- Documentation set (`ARCHITECTURE`, `API`, `DEPLOYMENT`, `RUNBOOK`, checklist).
